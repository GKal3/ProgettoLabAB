BookRecommender è un'applicazione client-server sviluppata in Java
che permette agli utenti di ricevere suggerimenti di lettura personalizzati,
lasciare valutazioni sui libri e cercare titoli in base a diversi criteri.
Il progetto è stato realizzato come parte del corso di Laboratorio B.

++REQUISITI++
Java JDK 17 + , sistema operativo Windows

++AVVIARE L'APPLICAZIONE++
Doppio click sui jar eseguibili o tramite linea di comando digitare, in questo ordine:
java -jar server-1.0-jar-with-dependencies.jar
java -jar client-1.0-jar-with-dependencies.jar
